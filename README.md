## Husky Track: AI-Powered Course Navigator
Husky Track is a React-based dashboard application built to streamline the course registration process for University of Washington students. By leveraging AWS Claude AI and a custom-aggregated dataset, it serves as an intelligent academic advisor, helping students navigate the complex UW Computer Science curriculum. Instead of manually going through multiple course and professor related information, this AI agent does it all for you in a chatbot format.
